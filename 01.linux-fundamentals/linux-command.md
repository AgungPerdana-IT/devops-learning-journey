# Linux Fundamentals


##  System Information
1. `whoami` - Check current user
2. `uptime` - Check system uptim

## File and Directory Management
3. `ls` - List files and directories in the current location
4. `cd /path/to/directory` - Changes the working directory
5. `pwd` - Prints the current working directory
6. `mkdir new_folder` - Creates a new directory
7. `rmdir empty_folder` - Removes an empty directory
8. `rm file.txt` - Deletes a file
9. `rm -r folder` - Deletes a folder and its contents
10. `cp file2.txt file3.txt` - Copies a file
11. `cp -r dir1 dir2` - Copies a directory recursively
12. `mv old_name new_name` - Moves or renames a file or directory

## File Viewing and Editing
13. `cat file.txt` - Displays file content
14. `tac file.txt` - Displays file content in reverse order
15. `less file.txt` - Opens a file for viewing with scrolling support.
16. `more file.txt` - Similar to `less`, but only moves forward
17. `head -n 10 file.txt` - Displays the first 10 lines of a file
18. `tail -n 10 file.txt` - Displays the last 10 lines of a file
19. `nano file.txt` - Opens a simple text editor
20. `vi file.txt` - Opens a powerful text editor
21. `echo 'Hello" > file.txt` - Writes text to a file, overwriting existing content
22. `echo 'Hello' >> file.txt` - Appends text to a file without overwriting
