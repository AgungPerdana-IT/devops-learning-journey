# Git Workflow

## Clone Repository
### 1.Clone the repository
```
git clone https://github.com/AgungPerdana-IT/devops-learning-journey.git
cd devops-learning-journey
```

Explanation :
- Download the repository
- Creates a local copy on your machine

## 2. Create a Feature Branch
Create a new branch for your work<br>
`git checkout -b feature/linux-fundamentals`

Explanation :
- `-b` creates a new branch
- You switch to that branch immediately

Check current branch :<br>
`git branch`

Example output:
```
feature/linux-fundamentals
main
```

## 3. Make Changes
Edit or add files to the project
Example :
```
linux-fundamentals/linux-fundamentals
```

## 4. Check Changes
Check repository status
```
git status
```

Example output:
```
modified: linux-fundamentals.md
```

## 5. Add Changes to Staging Area
Add files before committing
Add specific file :<br>
```
git add linux-fundamentals.md
```
Add all changes :
```
git add .
```

## 6. Commit Changes
Save changes with a message
```
git commit -m "add linux fundamentals commands"
```
Good commit message should be short and descriptive

## 7. Push Branch to GitHub
Upload the branch to Github<br>
```
git push origin feature/linux-fundamentals
```
Explanation :
- `origin` - remote repository
- `feature/linux-fundamentals` - branch name

## 8. Merge Changes into Main Branch
Switch to main branch :<br>
```
git checkout main
```
Pull latest updates :<br>
```
git pull
```
Merge feature branch :<br>
```
git merge feature/linux-fundamentals
```
Push changes :
```
git push origin main
```

## 9. Delete Feature Branch (Optional)
Delete local branch :
```
git branch -d feature/linux-fundamentals
```
Delete remote branch :
```
git push origin --delete feature/linux-fundamentals
```

## Example Workflow Summary
Typical development workflow :
```
1. Create branch
2. Make changes
3. git add
4. git commit
5. git push
6. merge into main
```


Git Commands Used in This Project
`git clone` - Copy repository from GitHub
`git checkout -b` - Create new branch
`git status` Show repository status
`git add` - Add files to staging
`git commit` - Save changes
`git push` - Upload changes to GitHub
`git merge` - Merge branches
