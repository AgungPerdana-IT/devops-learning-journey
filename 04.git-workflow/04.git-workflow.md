# Git Workflow

## Clone Repository
### 1.Clone the repository
```
git clone https://github.com/AgungPerdana-IT/devops-learning-journey.git
cd devops-learning-journey
```

Explanation :
- Download the repository
- Creates a local copy on your machine

## 2. Create a Feature Branch
Create a new branch for your work
`git checkout -b feature/linux-fundamentals`

Explanation :
- `-b` creates a new branch
- You switch to that branch immediately

Check current branch :
`git branch`

Example output:
```
feature/linux-fundamentals
main
```

## 3. Make Changes
Edit or add files to the project
Example :
```
linux-fundamentals/linux-fundamentals
```

## 4. Check Changes
Check repository status<br>
`git status`

Example output:
```
modified: linux-fundamentals.md
```
