# Git Workflow

## Clone Repository
### Clone the repository
1. `git clone https://github.com/AgungPerdana-IT/devops-learning-journey.git`
2. `cd devops-learning-journey`

Explanation :
- Download the repository
- Creates a local copy on your machine

## Create a Feature Branch
### Create a new branch for you work
3. `git checkout -b feature/linux-fundamentals`

Explanation :
- `-b` creates a new branch
- You switch to that branch immediately

### Check current branch :
4. `git branch`

Example output:
- feature/linux-fundamentals
- main

## Make Changes
### Edit or add files to the project
Example
```
linux-fundamentals/linux-fundamentals
```

## Check Changes
### Check repository status
6. `git status`

Example output:
- modified: linux-fundamentals.md
